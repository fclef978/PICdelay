使い方

基本的にプログラムの指示に従ってください。結果はコマンドライン上とresult.asmに出力されます。
計算できるのは単ループモードが2[us]〜1[ms]、二重ループモードが100[us]〜500[ms]、三重ループモードが1[ms]〜100[s]です。(標準設定の場合)

setting.iniについて
setting.iniは設定ファイルです。iniの形式ですが、ちゃんと読んでいるわけではないので、スペルや順番を変えないでください。

SystemClock(Hz)はPICのクロックです。PIC16F84Aを例にとると、CPI(一命令実行するのに必要なクロック数)が4なので、発振器が20[MHz]だとすると、周波数/CPIで5[MHz]です。
ただし、現段階では5[MHz]以外はサポートしていません。今後のサポートも未定です。特に、クロック周期が割り切れないような数になる周波数はバグります。
SubroutineLabelはサブルーチンの名前(ラベル)です。呼び出し時のラベル名を設定できます。
RegisterNameはカウントに使うレジスタの名前です。多重ループ時は、外側のループから1,2,3と割り当てられます。
LoopLabelはサブルーチン内でのループに使うラベルです。多重ループ時は、外側のループから1,2,3と割り当てられます。
